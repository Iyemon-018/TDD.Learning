# ボウリングゲーム
このプロジェクトは TDD を実践するための題材「ボウリングゲーム」を C# で実装するものです。

## 参考資料
- [Bowling Game Kata](https://kata-log.rocks/bowling-game-kata)
- [Bowling Game Kata in C\# Adapted](https://www.slideshare.net/mdclement/bowling-game-kata-in-c-adapted)

## ボウリングのルール
ゲームは10フレームで構成されています。各フレームでプレイヤーは2回の出目で10本のピンを倒します。フレームのスコアは、倒したピンの数、ストライクとスペアのボーナスの合計となります。
スペアとは、プレイヤーが2回の出目で10本のピンをすべて倒した場合です。そのフレームのボーナスは、次の出目で倒したピンの本数です。
ストライクとは、プレイヤーが最初のロールで10本のピンをすべて倒した場合です。その後、1回の出目でフレームを完成させる。そのフレームのボーナスは、次の2本の賽の目となる。
10フレーム目でスペアまたはストライクを出したプレーヤーは、フレームを完成させるために追加のボールを転がすことができる。ただし、10コマ目に転がすことができるボールは3個までとする。

### 必要条件
以下の2つのメソッドを持つクラスGameを作成します。

1. `void roll(int)`はプレイヤーがボールを転がすたびに呼び出される。引数は倒したピンの本数である．
2. `int score()`はそのゲームの総得点を返す。

※このルールは[Bowling Game Kata](https://kata-log.rocks/bowling-game-kata)を翻訳したものです。
